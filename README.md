# Checked Out

A 3-tier web application for teaching purposes, demonstrating full-stack development with modern technologies.

## Tech Stack

### Frontend
- **React** - UI framework for building the client-side application

### Backend
- **Node.js** - JavaScript runtime environment
- **Express** - Web application framework for building the REST API

### Database
- **MySQL** - Relational database management system

## Architecture

This application follows a 3-tier architecture:

1. **Presentation Layer** - React frontend providing the user interface
2. **Application Layer** - Node.js/Express backend handling business logic and API endpoints
3. **Data Layer** - MySQL database for persistent data storage
